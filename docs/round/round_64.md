좋아. **R10 Loop 2까지 끝났으니, 이번은 R11 Loop 2 — 정책·지정학·재난·이벤트**로 넘어간다.

R11은 다시 강조하지만 **좋은 종목 발굴 라운드라기보다 false-positive 방어 라운드**다. Theme Tag Map에서도 남북경협, DMZ, 개성공단, 금강산 관광, 황사·마스크, 엠폭스, 코로나 진단, 초전도체, 맥신, 그래핀, 양자기술 같은 태그는 대부분 `Event/Red`, `Watch/Red`, `SPECULATIVE_SCIENCE_THEME`로 분류되어 있고, Stage 3-Green보다 RedTeam과 4B 방어가 중요하다고 정리되어 있다.

서생원식으로 보면 R11의 기준은 아주 간단하다. **뉴스가 큰가?**가 아니라 **뉴스가 실제 계약·예산·매출·EPS/FCF 체급 변화로 넘어갔는가?**다. 이벤트는 Stage 1을 만들 수 있지만, Stage 3-Green을 만들려면 실제 수익화 증거가 있어야 한다.

# 4. 성공사례

## 4-1. Bavarian Nordic — 엠폭스 이벤트가 정부계약·가이던스로 승격되는 경우

Bavarian Nordic은 R11에서 보기 드문 `EVENT_TO_CONTRACT_ESCALATION` 후보로 둔다. 회사는 미국 정부의 Jynneos 동결건조 제형 추가 계약 옵션 9,700만 달러를 받았고, 이 계약을 반영해 2026년 매출 가이던스를 50억~52억 덴마크크로네에서 55억~57억 덴마크크로네로, EBITDA margin 전망을 25%에서 약 28%로 올렸다. 즉 단순 엠폭스 뉴스가 아니라 **정부 stockpile 계약 + 매출 가이던스 상향 + EBITDA margin 상향**으로 승격된 사례다. ([Reuters][1])

```text
가격경로 1차 판정:
EVENT_TO_CONTRACT_SUCCESS_CANDIDATE

좋은 점:
- 정부계약
- stockpile 수요
- 매출 가이던스 상향
- EBITDA margin 상향
- FDA 승인 제형과 정부 비축 연결

주의:
- 전염병 수요는 반복성이 약할 수 있음
- stockpile 계약이 반복되는지 확인 필요
- outbreak 정상화 시 EPS 정상화 가능
```

**Loop 2 교정**

```text
EVENT_DISEASE_PEST_DEMAND:
뉴스만 있으면 Event/Red.
정부계약 + 매출 가이던스 + 반복 stockpile 가능성이 있으면 Stage 2 후보.
Stage 3는 반복 조달과 FCF 확인 전까지 제한.
```

## 4-2. Bavarian Nordic 2024 mpox 가격반응 — event premium의 전형

Bavarian Nordic은 2024년에도 mpox 관련 수요와 주문으로 주가가 급등한 사례가 있다. Investopedia는 2024년 2분기 실적과 44만 도즈 유럽 주문 보도 이후 주가가 12% 뛰었고, 당시 mpox 관련 기대 속에서 한 달간 45% 이상 상승했다고 정리했다. 이건 R11의 전형적인 흐름이다. **전염병 뉴스 → 단기 주가 급등 → 실제 정부 주문·가이던스가 붙는지 확인**으로 가야 한다. ([Investopedia][2])

```text
가격경로 1차 판정:
EVENT_PREMIUM_WITH_CONTRACT_VALIDATION_NEEDED

의미:
전염병 관련주 급등은 Stage 1 event premium.
정부계약·가이던스·반복 비축이 붙으면 Stage 2.
그렇지 않으면 one-off event.
```

## 4-3. 우크라 통신망 복구 — 재건이 실제 financing으로 승격되는 경우

우크라 재건은 대부분 Event/Watch지만, 실제 금융기관 자금과 인프라 자산이 붙으면 Stage 2 후보가 된다. EBRD와 IFC는 우크라이나의 Lifecell과 Datagroup-Volia를 결합한 신규 통신회사에 총 4.35억 달러를 제공하기로 했고, 이는 네트워크 속도·커버리지·회복력을 높이는 통신 인프라 투자로 설명된다. 이 사례는 재건 테마가 **실제 외국인 투자·인프라 자산·운영회사**로 구체화된 경우다. ([Reuters][3])

```text
가격경로 1차 판정:
REAL_RECONSTRUCTION_FINANCING_CANDIDATE

좋은 점:
- EBRD·IFC financing
- 실제 통신 인프라 자산
- 인수·합병 완료 회사
- network resilience 개선 목적
- 보증 구조 존재

주의:
- 한국 관련주로 직접 매핑하려면 실제 수주·계약·매출 연결 필요
- 재건 선언이나 MOU와 구분해야 함
```

**Loop 2 교정**

```text
GEOPOLITICAL_RECONSTRUCTION:
재건회의·지원선언 = Stage 1.
financing + 참여기업 + 자산 + 계약 = Stage 2.
다년 매출·마진·수주잔고 확인 전 Stage 3 금지.
```

## 4-4. 폭염·냉각·전력망 — 이벤트가 다른 대섹터 구조증거로 넘어가는 경우

폭염 자체는 `CLIMATE_DISASTER_EVENT`지만, 전력망·냉각·수요반응·VPP로 연결되면 R1/R2/R3와 교차하는 구조증거가 될 수 있다. 2025년 독일 사례 연구는 가정용 이동식 에어컨 보급률이 19%에서 35%로 올라가는 폭염 시나리오에서 피크 전력수요가 12.9GW 이상 증가할 수 있고, 오후 피크가 태양광 출력 저하 시간과 겹쳐 전력계통 안정성 부담을 키울 수 있다고 분석했다. ([arXiv][4])

또 뉴욕에서는 폭염 피크 시간대 창문형 에어컨 부하를 줄이기 위해 임차인에게 plug-in battery를 제공하는 시범 프로그램이 진행되고 있고, 이 프로그램은 200kW flexible capacity에서 2MW 이상·1,000가구 이상으로 확장될 예정이라고 보도됐다. 이건 폭염 테마가 단순 에어컨 이벤트가 아니라 **수요반응·분산배터리·VPP**로 연결될 수 있음을 보여준다. ([AP News][5])

```text
가격경로 1차 판정:
CLIMATE_EVENT_TO_GRID_INFRA_WATCH

좋은 점:
- 폭염 → 전력피크
- 에어컨 → grid stress
- battery/VPP → 실제 프로그램
- R1 전력망, R2 냉각, R3 ESS와 교차 가능

주의:
- 폭염 자체는 계절성 이벤트
- 개별 종목은 실제 계약·매출·OPM 필요
```

**Loop 2 교정**

```text
CLIMATE_DISASTER_EVENT:
폭염·황사는 Stage 1 event.
전력망 투자, 냉각 수주, ESS/VPP 계약이 붙으면 다른 대섹터의 Stage 2로 이관.
```

```text
NORTH_KOREA_POLICY_EVENT_RED_BIAS

남북경협 기대
```

```text
NORTH_KOREA_POLICY_EVENT:
```

## 5-2. LK-99 — speculative science hard counterexample

LK-99는 R11에서 가장 중요한 과학 테마 반례다. 2023년 LK-99가 상온·상압 초전도체라는 preprint가 나오며 전 세계적 관심을 받았지만, 여러 독립 재현 연구는 초전도성을 확인하지 못했다. 한 arXiv 연구는 LK-99 샘플이 실온에서 초전도성을 보이지 않았고 SQUID magnetization 측정에서도 실온 초전도 신호가 없었다고 보고했다. ([arXiv][7])

또 다른 재현 연구는 원 논문에서 초전도 증거처럼 보인 전기·자기 이상 현상을 Cu₂S 불순물의 구조전이에 의한 것으로 설명했다. 즉 LK-99는 **preprint·SNS·영상 기반 테마가 실제 재현·상용화·매출로 넘어가지 못한 대표 사례**다. ([arXiv][8])

```text
가격경로 1차 판정:
SPECULATIVE_SCIENCE_FAILURE_HARD_4C

교훈:
논문·preprint·영상
≠ 제품
≠ 고객계약
≠ EPS/FCF

Stage 3 금지 조건:
- 재현 실패
- peer review 없음
- 상용 제품 없음
- 고객사 없음
- 매출 없음
```

**Loop 2 교정**

```text
SPECULATIVE_SCIENCE_THEME:
기본값 Red.
Stage 2가 되려면 independent replication + prototype customer validation + 계약이 필요.
```

전염병 진단은 EPS가 폭발할 수 있지만 지속성이 약하다. Abbott은 2025년 3분기 매출이 시장 예상에 소폭 못 미쳤고, diagnostics segment 매출은 COVID-19 및 당뇨 검사 수요 약화 등으로 6.6% 감소했다. 이 사례는 진단키트·전염병 수요가 정상화되면 EPS가 빠르게 꺾일 수 있음을 보여준다. ([Reuters][9])

```text
가격경로 1차 판정:
DIAGNOSTICS_ONE_OFF_NORMALIZATION_4C

교훈:
전염병 진단 매출
≠ 구조적 Green

Green 조건:
- 설치기반
- 다양한 검사 메뉴
- 비전염병 반복검사

- one-off 수요 이후에도 FCF 유지
```

## 5-4. 폭염·황사·재난 테마의 계절성

폭염과 황사, 미세먼지, 재난복구는 주가를 움직일 수 있지만 대부분은 계절성·one-off 수요다. 폭염의 경우 전력망·VPP·냉각 투자로 연결되면 구조증거가 될 수 있지만, 단순 에어컨·마스크·공기청정기 수요만으로는 Stage 3-Green이 아니다. 폭염 전력수요 연구와 뉴욕 배터리 프로그램은 “계절성 뉴스”가 실제 grid-management program으로 승격될 수 있는 경로를 보여주지만, 개별 기업 점수화는 실제 계약·매출·OPM이 있어야 한다. ([arXiv][4])

```text
기후 이벤트
≠ 자동 Green

Stage 2 조건:
- 전력망 투자
- 냉각 수주
- VPP/배터리 프로그램
- 실제 매출
- 반복 계약
```
