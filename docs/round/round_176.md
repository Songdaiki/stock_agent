좋아. **R4 Loop 11까지 끝났으니, 이번은 R5 Loop 11 — 소비재·유통·브랜드**다.

이번 R5는 조건을 더 빡세게 간다.

```text
Loop 11 추가 조건:
1. 국장 종목 중심
2. 기존 핵심 사례였던 삼양식품·APR·쿠팡·Whirlpool·Shein/Temu 중심 반복 금지
3. 이번에는 실리콘투, 달바글로벌, CJ/올리브영, 농심, 코스맥스·한국콜마, 아모레퍼시픽, F&F류를 중심으로 재분류
4. 주가가 나중에 크게 오른 종목을 Stage 3 전에 잡는지 확인
5. 이미 상승 릴레이가 너무 많이 진행됐거나 sell-through·OPM이 못 따라오면 Stage 4B/4C로 식히는지 확인
```

R5는 라면, K푸드, 음식료, 건강기능식품, 편의점, 홈쇼핑, 이커머스, 콜드체인, 화장품 브랜드, 화장품 OEM·ODM, 의류 브랜드·OEM, 유아용품, 생활가전까지 흡수한다. 이 섹터는 **수출·반복소비·OPM·채널 확장**이 있으면 Green 가능성이 있지만, 단일제품 유행, 재고, 매출채권, 리콜, 원가 상승은 4C 후보가 된다.

서생원식으로 보면 R5의 질문은 “브랜드가 유명한가?”가 아니다. 핵심은 **반복소비·ASP·해외 채널·sell-through·재주문·OPM·FCF가 EPS 체급을 바꾸고, 시장이 아직 과거 내수 소비재 프레임으로 보고 있는가**다. Viral, 입점, 사용자 수, GMV는 Stage 1~2이고, 숫자로 반복성이 확인돼야 Stage 3다.
