좋아. **R1 Loop 11까지 끝났으니, 이번은 R2 Loop 11 — AI·반도체·전자부품**이다.

이번 Loop 11은 조건을 바꾼다.

```text
Loop 11 추가 조건:
1. 국장 종목 중심
2. 기존 핵심 성공/반례였던 SK하이닉스·삼성 HBM4·Supermicro·Broadcom·Cisco 등은 중심 사례에서 제외
3. “나중에 오른 종목을 뒤늦게 좋다고 하는 것”이 아니라,
   상승 전 Stage 3 후보로 포착 가능한지 확인
4. 이미 너무 오른 종목은 Stage 4B로 식히는지 확인
5. 계약·수주·매출·OP/EPS·고객명·price-path가 없는 테마는 Stage 3 금지
```

R2는 HBM, CXL, 시스템반도체, 뉴로모픽, 반도체 장비·소재, PCB, 유리기판, MLCC, OLED, 클린룸, 스마트폰 부품, 카메라, 무선충전까지 흡수한다. 이 대섹터에서 HBM은 Green 가능성이 높지만, 장비·소재·PCB·테스트·AI칩 테마는 **실제 고객, 계약, 출하, OP/EPS 상향, 가격경로**를 통과해야 한다.

서생원식으로 보면 R2의 핵심은 “AI 수혜주냐?”가 아니라 **AI 인프라 병목이 EPS/FCF 체급 변화로 바뀌고, 시장이 과거 부품주·장비주·테마주 프레임을 버리며 리레이팅하는가**다. 즉 HBM 장비, 테스트 소켓, AI 서버 PCB, 시스템반도체, NPU 모두 같은 질문을 통과해야 한다.

한미반도체는 성공사례이면서 동시에 경고 사례다. WSJ 보도는 Micron 공급 가능성에 주가가 최대 22% 올랐다고 했지만, 같은 보도는 한미와 Micron이 해당 media reports에 대해 즉시 답변하지 않았고, Micron deal은 “if finalized”의 성격이었다고 정리한다. 즉 **SK하이닉스 계약은 강한 Stage 2 근거지만, Micron 건은 확정계약 전까지 Stage 3로 올리면 안 된다**. ([월스트리트저널][1])

```text
case_type:
MOU_OR_REPORT_NOT_CONTRACT
+
DISCLOSURE_CONFIDENCE_CAPPED
+
4B_RALLY_WATCH
```

**점수표 반영**

```text
확정 고객계약:
Stage 2 가점

보도·협의:
Stage 2 보조 가점

확정계약 미공시:
Stage 3 cap

4B watch 즉시 부착
```

### 5-2. 이수페타시스 487% 급등 — 좋은 구조여도 주가가 먼저 5배 가면 Stage 4B

이수페타시스는 AI 서버 PCB 구조가 맞았을 수 있지만, 487% stock surge가 이미 발생한 뒤에는 신규 점수표가 Stage 3로 계속 밀면 안 된다. 급등 뒤에는 OP/EPS revision이 더 따라오지 않으면 4B 또는 4C로 내려야 한다. ([위키백과][2])


```text

또는 Stage3 이후 252D MFE +150% 이상
또는 주가 400% 이상 상승

이면 신규 진입 점수는 자동 감점.
```

**점수표 반영**

```text
price_path_400pct_plus:

```

# R2 Loop 11 결론

이번 R2 Loop 11의 핵심은 이거다.

```text
국장 R2는 HBM 본체만 보는 라운드가 아니다.
HBM 장비, AI 서버 PCB, 테스트 소켓, 시스템반도체 option,
국산 AI칩 관련주까지 모두 보되,
실제 계약·고객·OP/EPS·가격경로로 걸러야 한다.
```

```text
Stage 3로 조기 포착해야 했던 후보:
한미반도체
이수페타시스
리노공업

단, 이미 크게 오른 구간은 Stage 4B 감시:
한미반도체 급등 이후
이수페타시스 487% stock surge 이후
리노공업 AI boom +70% 이후

Stage 1~2지만 Green 금지:
DB하이텍 정책 foundry / ReRAM option
리벨리온·사피온 관련주
온디바이스 AI 관련주
AI칩 private valuation 관련주

4C 또는 hard watch:
미확정 Micron 보도만으로 Stage 3 부여
삼성 파업·생산차질 overlay
고객·계약·매출 연결 없는 AI칩 관련주
주가 300~500% 급등 후 OP/EPS 추격 실패
```

**R2 Loop 11 점수정규화의 핵심 문장:**

> AI·반도체·전자부품은 “HBM”, “AI 서버”, “PCB”, “테스트 소켓”, “NPU”, “온디바이스 AI”라는 이름이 아니라 **고객명, 계약금액, 계약기간, 장비 출하, 반복수주, OP/EPS 상향, 고객다변화, price-path MFE/MAE, valuation room, disclosure confidence**로 봐야 한다.
> 이번 Loop 11부터는 점수표가 **한미반도체·이수페타시스·리노공업 같은 대폭 상승 후보를 Stage 3 초입에서 잡을 수 있었는지**, 그리고 **상승 릴레이가 과열되면 Stage 4B로 식히는지**를 의무 검증한다.

다음 순서는 **R3 — 2차전지·전기차·친환경 Loop 11**, 역시 **국장 중심 + 기존 사례 제외 + 실제 가격경로 검증 강화**로 간다.
