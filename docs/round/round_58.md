좋아. **R4 Loop 2까지 끝났으니, 이번은 R5 Loop 2 — 소비재·유통·브랜드**로 넘어간다.

이번 R5 2회차의 목표는 더 선명하다.

```text
수출 소비재 구조적 리레이팅
≠ 단일 제품 viral

K뷰티 글로벌 채널 확장
≠ TikTok 유행만으로 Green

이커머스 scale
≠ FCF·규제·데이터 보안 통과

렌탈 반복매출
≠ 생활가전 hardware cycle

의류·fast fashion 성장
≠ 재고·IP·규제 리스크 면제
```

R5는 Theme Tag Map 기준으로 라면, K푸드, 음식료, 편의점, 홈쇼핑, 이커머스, 콜드체인, 화장품 브랜드, 화장품 OEM/ODM, 화장품 원재료, 의류, 키즈, 생활가전을 흡수하는 대섹터다. 이 지도에서도 K푸드·K뷰티는 Green 가능, 이커머스·홈쇼핑·의류·생활가전은 Watch 중심으로 분리되어 있다.
그리고 R5에서도 Checkpoint 20 원칙은 그대로 적용된다. 수출액, 계약금액, OP YoY, 재고, 매출채권, 공급계약, 채널 입점 같은 값은 실제 공시·리포트·기사에서 확인될 때만 쓰고, 빈칸을 추정해서 채우면 안 된다.
서생원식으로도 이 라운드는 “인기 있는 브랜드”가 아니라 **EPS/FCF 체급 변화와 밸류에이션 리레이팅이 동시에 일어나는 구조**를 찾는 라운드다.

# R5 Loop 2. 소비재·유통·브랜드

## 1. 이번 라운드 대섹터

```text
R5 = 소비재·유통·브랜드
Loop 2 목표 = 수출·반복소비·채널확장형 Green과 viral/event형 false-positive를 분리
```

이번 회차의 핵심 질문은 이거다.

```text
이 브랜드는 정말 반복 소비와 채널 확장으로 EPS/FCF 체급이 바뀌는가?
아니면 단일 제품 viral, 재고 밀어내기, 상장 기대, 사용자 수, 점포 수, 이벤트성 수요인가?
```

R5에서 오판이 가장 자주 나오는 지점은 아래다.

```text
1. 수출 증가를 sell-through 검증 없이 Green으로 처리
2. TikTok viral을 반복소비로 착각
3. 채널 입점을 실제 재주문·OPM 개선으로 착각
4. 이커머스 매출 성장을 FCF 개선으로 착각
5. 렌탈 계정 없이 생활가전 hardware를 반복매출로 오판
6. 의류·fast fashion 성장을 재고·IP·규제 리스크 없이 점수화
```
